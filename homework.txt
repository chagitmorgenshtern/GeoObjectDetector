מערכת זיהוי אובייקטים גיאוגרפיים בתמונה אווירית
רקע
נדרש לזהות את מיקומם הגיאוגרפי של מבנים בתמונה אווירית (לווין או רחפן). בתמונה יש מספר מבנים ונדרש להחזיר עבור כל מבנה את מרכזו ו\או את מרכז קו המתאר הארוך ביותר בגג.
דרישות
1. כתוב תוכנית ב־C++ שקוראת קובץ תמונה.
2. השתמש בספריית עיבוד תמונה כדי לזהות את המבנים\קווי מתאר.
3. לכל אובייקט שזוהה, המר את מיקומו הפיקסלי למיקום גיאוגרפי (Latitude, Longitude) 
4. הפלט יהיה JSON או קובץ טקסט שבו רשימת האובייקטים שנמצאו עם מיקומם (lat/lon).
5. הפתרון צריך לרוץ כ־ Web Server ולהנגיש API שמאפשר להפעיל את פונקציית הזיהוי על תמונה וקלט גיאוגרפי.
